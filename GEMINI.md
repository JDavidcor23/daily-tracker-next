# Daily Tracker

Personal daily tracker built with React + TypeScript + Tailwind CSS + Supabase. Logs nutrition, training, study, and mental health per day. Includes a Todo manager, Google Fit integration, and PDF report generation.

## Agents

- **constants-enforcer** — No magic numbers or strings; always use named constants.
- **file-structure-enforcer** — Folder structure, naming conventions, and barrel exports.
- **styles-enforcer** — Tailwind CSS only; no inline styles, no Styled Components.
- **react-patterns-enforcer** — Separation of layers, Context vs props drilling, Provider placement.
